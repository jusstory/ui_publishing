접근성 키 맵핑: up,left = 이전 단계, down,right = 다음 단계, home = 최저, end= 최대
인자값: container_id, vert, min, max, inc, jump, showVals, range, val1, val2

가로 스크롤바

감싸는 영역 sliderWrap 클래스, 가로 스크롤바 클래스 hslider,
슬라이더 호출시 id선언

//가로 스크롤바 
TNUI.module.sliderUi('sr1', false, 0, 100, 5, 10, true, false, 30);
<div class="sliderWrap">
    <div class="hslider" id="sr1"></div>
</div>

가로 스크롤바 범위형

감싸는 영역 sliderWrap 클래스, 가로 스크롤바 클래스 hslider,
슬라이더 호출시 id선언

//가로 스크롤바 범위형
TNUI.module.sliderUi('sr2', false, 190000, 200800, 1, 10, true, true, 195000, 200000);
<div class="sliderWrap">
    <div class="hslider" id="sr2"></div>
</div>

세로 스크롤바 범위형

감싸는 영역 sliderWrap 클래스, 가세 스크롤바 클래스 vslider,
슬라이더 호출시 id선언

//세로 스크롤바 범위형
TNUI.module.sliderUi('sr3', true, 1, 10, 1, 2, true, true, 3, 5);
<div class="sliderWrap">
    <div class="vslider" id="sr2"></div>
</div>
